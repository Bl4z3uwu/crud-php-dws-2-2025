# PHP CRUD - Web Site Development 2 (DWS-2)

Este repositório contém o código-fonte de um **CRUD** (Create, Read, Update, Delete) desenvolvido em **PHP**, como parte das atividades da disciplina de **Desenvolvimento de Web Sites 2** do curso técnico em informática.

## Objective

This project aims to implement the basic operations of a CRUD system using PHP, covering essential web development concepts such as database handling, code structure, and programming best practices.

## Technologies Used

- **PHP**
- **MySQL**
- **HTML/CSS**
- **Bootstrap**
