# PHP CRUD - Web Site Development 2 (DWS-2)

This repository contains the source code for a **CRUD** (Create, Read, Update, Delete) system developed in **PHP**, as part of the coursework for the **Web Site Development 2** subject in the Technical Course in Informatics.

## Objective

This project aims to implement the basic operations of a CRUD system using **PHP**, covering essential web development concepts such as database handling, code structure, and programming best practices.

## Technologies Used

- **PHP**
- **MySQL**
- **HTML/CSS**
- **Bootstrap**
