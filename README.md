# CRUD em PHP - Disciplina Desenvolvimento de Web Sites 2 (DWS-2)

Este repositório contém o código-fonte de um **CRUD** (Create, Read, Update, Delete) desenvolvido em **PHP**, como parte das atividades da disciplina de **Desenvolvimento de Web Sites 2** do curso técnico em informática.

## Objetivo

O projeto visa implementar as operações básicas de um sistema CRUD utilizando **PHP**, abordando conceitos essenciais do desenvolvimento web como manipulação de banco de dados, estruturação de código, e boas práticas de programação.

## Tecnologias Utilizadas

- **PHP**
- **MySQL**
- **HTML/CSS**
- **Bootstrap**
